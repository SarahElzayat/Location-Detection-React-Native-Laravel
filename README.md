## Installation

### Frontend

```bash
npm install
```

### Backend

```bash
npm install ngrok

```

```bash
composer install 
```

## Running

### Frontend

```bash
npx expo run
```

### Backend

```bash
php artisan:migrate
```

```bash
php artisan serve
```

```bash
ngrok http 8000 #(or the used port number) 
```
